# YOLOv5를 이용한 웹앱 프로그램 // 250409
yolov5.pt 원형 모델을 이용하여 objecty detect를 하는 파이썬 프로그램을 하였고
파이썬 코드를 index.html로 바꾸어 달라고 github copliot을 이용해서 진행 하였습니다.

# 아래의 프로그램을 실행시키기 위한 라이브러리 설치
```
pip install -r requirements.txt
```
